# Cyber Threat Detection using Machine Learning

##  Project Overview

This project focuses on detecting cyber threats using Machine Learning techniques.
The model is trained on the NSL-KDD dataset to classify network traffic into normal and attack categories.

---

##  Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib
* Seaborn

---

##  Algorithm Used

* Multi-Layer Perceptron (MLPClassifier)
* A type of Neural Network used for classification

---

##  Dataset

* NSL-KDD Dataset
* Contains normal and attack network traffic

---

##  How to Run

1. Install dependencies:
   pip install pandas numpy matplotlib seaborn scikit-learn

2. Run the code:
   python cyber_threat_detection.py

---

##  Results

* Model Accuracy: ~77%
* Confusion Matrix used for evaluation
* Classification report generated

---

##  Conclusion

The model successfully detects cyber threats with good accuracy and demonstrates the effectiveness of machine learning in cybersecurity.
